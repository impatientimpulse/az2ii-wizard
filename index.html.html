How to install

1

Install Open Wizard in Kodi

In Kodi: Add-ons → Install from zip → navigate to the Open Wizard zip, or install from the Kodi repository under Program add-ons.

2

Add this build source

Open Open Wizard → Wizard tab → Add Wizard → paste the URL above. The build will appear in the list.

3

Install the build

Select AZ2 ImpatientImpulse Edition – Full Build → Install. Open Wizard handles everything: skin, addons, and settings.

4

Add your TMDb API key

After install: Add-ons → TMDb Helper → Settings → paste your free key from themoviedb.org/settings/api

5

Authorize Trakt

Add-ons → Trakt → Authorize → visit trakt.tv/activate and enter the code. This unlocks your Continue Watching and Watchlist rows.
